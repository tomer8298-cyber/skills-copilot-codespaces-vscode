# GitHub Copilot Guide with VS Code and Codespaces

_Get started using GitHub Copilot in VS Code Codespaces._

## Welcome

- **Who is this for**: Developers at any experience level looking to accelerate their code workflow.
- **What you'll learn**: How to use GitHub Copilot to write, complete, and improve code.
- **What you'll build**: JavaScript functions guided by GitHub Copilot suggestions.
- **Prerequisites**: Basic coding principles and familiarity with VS Code.
- **How long**: This exercise takes less than one hour to complete.

## What is GitHub Copilot?

GitHub Copilot is an AI pair programmer that helps you write code faster and with less effort. It draws context from comments and code, and suggests individual lines and whole functions instantly.

## Steps

### Step 1: Open a Codespace

1. Click the **Code** button on this repository.
2. Click the **Codespaces** tab.
3. Click **Create codespace on main**.

### Step 2: Use Copilot to write a function

1. Open the `skills.js` file.
2. Start typing a comment describing what you want to build, for example:
   ```js
   // function to reverse a string
   ```
3. Press **Enter** and wait for Copilot to suggest code.
4. Press **Tab** to accept the suggestion.

### Step 3: Use Copilot for code completion

1. In `skills.js`, type the beginning of a function:
   ```js
   function calculateSum(arr) {
   ```
2. Copilot will suggest a completion — press **Tab** to accept it.

### Step 4: Explore Copilot Chat

1. Open GitHub Copilot Chat from the sidebar.
2. Ask a question about your code, for example:
   - "Explain this function"
   - "Write a unit test for this function"
   - "How can I improve this code?"
